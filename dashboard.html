Claude Code 사용량

작업 실행 횟수
-
Claude가 파일 읽기·수정·검색 등 작업을 실행한 총 횟수
입력량
-
Claude에게 보낸 분량(질문·파일·맥락). 단위는 토큰
출력량
-
Claude가 만들어낸 답변 분량. 단위는 토큰
재활용한 분량
-
다시 안 읽고 재활용해 아낀 입력량(캐시)
예상 비용
-
사용량을 달러로 환산한 추정치(USD)
캐시 효율
-
읽은 것 중 재활용으로 싸게 처리한 비율. 높을수록 절약
스킬 사용
-
자동화 기능(스킬)을 실행한 횟수
외부 연동(MCP) 사용
-
외부 도구 연동(MCP)을 호출한 횟수

※ 토큰은 AI가 글을 처리하는 기본 단위예요(대략 단어 한두 조각). 숫자가 클수록 더 많이 처리했다는 뜻이고, 비용도 그만큼 커집니다.
